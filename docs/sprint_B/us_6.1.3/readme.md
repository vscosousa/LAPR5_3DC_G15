# 6.1.2 - As user, I want the application menu to adjust according to my role

## 1. Context

 `Description:` As user, I want the application menu to adjust according to my role so that it only presents me the options I may access.

 `Sprint:` This US makes part of the first stage of development of the integrative project of this semester, Sprint B.

`Objectives:` This user story aims to develop the application menu according to the user's role, so that the user can only access the options that are allowed to him.
